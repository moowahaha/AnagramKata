Variation on the popular anagram kata taken from:
http://codingdojo.org/cgi-bin/wiki.pl?KataAnagram

If you're not sure what an anagram is, here's some help:
http://en.wikipedia.org/wiki/Anagram

Rules:
 - Tests must be written first.
 - Do not use the anagram solver mentioned below!
 - Results must be [list of] single words.
 - Anagrams returned cannot be names.

If you're not sure about your anagram, check it here:
http://www.ssynth.co.uk/~gay/anagram.html

I have purposely not included a word list, that's part of
the challenge.

A sample passing phpunit test has been provided. To run it:
    $ phpunit test
